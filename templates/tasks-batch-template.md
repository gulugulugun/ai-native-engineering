## 任务依赖关系

> 用 ASCII DAG 标注任务间的前后依赖和可并行关系。
> 格式参考 mkt 团队实践：`T1.1, T1.2 (并行)` 表示可同时执行，`T2.1 ← T1.1` 表示 T2.1 依赖 T1.1 完成。
> 当前用于辅助人审查和 AI 串行执行时的排序判断；未来可作为 CLI Agent Teams 并行调度的输入。

```
示例：
T1.1, T1.2, T1.3 (并行)
    ↓
T2.1 ← T1.1, T1.3
T2.2 ← T1.2
    ↓
T3.1 ← T2.1, T2.2
```
